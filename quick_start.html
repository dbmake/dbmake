DBMAKE

A tool for schema and data migration. It provides migration capabilities similar to those implemented by Rails rake and Django South, however in is not tied to any framework, so it can be used with any language and technology.

Dependencies

3rd party software:

  • PostgresSQL server

Python packages:

  • See requirements.txt

To setup all dependencies run: sudo pip install -r requirements.txt
